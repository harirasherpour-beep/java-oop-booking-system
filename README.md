# Java Booking System

This project is a simple booking system developed in Java as part of my studies in Computer Science at NTNU.

## Features
- Create and manage reservations
- Store booking data in files
- Handle errors such as unavailable rooms
- Structured using object-oriented principles

## Technologies
- Java
- JavaFX
- Maven
- JUnit (testing)

## What I learned
- Object-oriented programming (OOP)
- File handling and data storage
- Error handling in applications
- Structuring larger applications using MVC principles

## How to run
1. Clone the repository
2. Open in IntelliJ or VS Code
3. Run the main application file
